# 📚 jQuery

Repositório desenvolvido para estudos direcionados em **jQuery**, abordando desde os conceitos fundamentais até manipulação de elementos, eventos, efeitos e animações.

O objetivo deste projeto é servir como material de consulta e prática durante o aprendizado da biblioteca.

---

## 📖 Conteúdo

Os exemplos estão organizados em arquivos HTML independentes, facilitando o estudo de cada tópico.

### ✔️ Seletores
- Selecionando elementos
- Navegação entre elementos
- Recuperando e manipulando atributos

### ✔️ Manipulação do DOM
- Inserindo elementos
- Removendo elementos
- Alterando conteúdo
- Manipulando CSS

### ✔️ Eventos
- Eventos básicos
- Eventos extras
- Método `on()`
- Método `off()`

### ✔️ Efeitos
- Show
- Hide
- Toggle
- Fade
- Slide

### ✔️ Animações
- Método `animate()`
- Controle de animações
- Encadeamento de efeitos

### ✔️ Sequência de execução
- Fluxo de execução do jQuery
- `$(document).ready()`

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/jquery.git
```

Acesse a pasta:

```bash
cd jquery
```

Instale as dependências:

```bash
npm install
```
